# IOAO Consulting — Portfolio

Personal portfolio site for IOAO Consulting.

Built with vanilla HTML, CSS, and JS — no framework, no build step, no dependencies beyond Google Fonts.

## Stack
- Single `index.html` file
- Deployed via GitHub Pages
- Custom domain: www.ioaoconsulting.com

## Local Development
Just open `index.html` in a browser. No server needed.

## Deployment
GitHub Pages from `main` branch. Every push to `main` auto-deploys.
